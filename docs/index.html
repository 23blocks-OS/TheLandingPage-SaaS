How It Works

1

Deploy Infrastructure

Use Terraform to deploy AWS resources (S3, CloudFront, Route53) in your account.

2

Setup Platform

Configure and deploy the Next.js platform with your database.

3

Create Projects

Use the dashboard or CLI to create projects and get unique subdomains.

4

Deploy Sites

Run 23blocks-deploy deploy and your site is live instantly.

# Install CLI
npm install -g @23blocks/deploy-cli

# Login
23blocks-deploy login

# Deploy your site
cd your-project
npm run build
23blocks-deploy deploy

# ‚úÖ Your site is live at https://yourproject.yourdomain.com
